FreeBSD / ppc Project


Introduction


The FreeBSD/ppc project pages contain information about the FreeBSD port to the PowerPC architecture. As with the port itself, these pages are still a work in progress.

Status


The FreeBSD/ppc port is stil a Tier 2 platform. That means it is not being fully supported by our security officer, release engineers and toolchain maintainers.

Latest News

  • 03 April, 2009: Support for the PowerPC G5 has been commiteed to SVN.
  • 03 March, 2008: Support for the Freescale PowerQUICC III MPC85XX family system-on-chip development boards has been commiteed into CVS.
  • 27 February, 2008: FreeBSD 7.0 is the first release to officially support the FreeBSD/ppc port.
